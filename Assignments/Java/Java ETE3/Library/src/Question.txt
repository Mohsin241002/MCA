You are taked with developing a java application to manage a small library database. The library databse stores details about books. including BookID(Primary Key), Title, Author, Genre and Price.
The application should connect to MySQL databse and allow the user to perform the following operation through a console-based menu.

JDBC connectivity through (MySQL Source/ DriverManager)
1. Create- Add a new book record.
2. Read - View all book records or search a specific book by title 
3. Update - Update the details of a book by its ID
4. Delete - Remove a book record by its ID
    (Minimum 6 records has to be inserted)
